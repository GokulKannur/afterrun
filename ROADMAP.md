# Roadmap

## Phase 4: Authentication
- User signup/login (Email + Password).
- JWT session management.
- Associate Jobs with Users.

## Phase 5: Billing
- Stripe integration.
- Subscription tiers (Free/Pro).
- Usage limits based on tier.

## Phase 6: Write UI
- Create/Edit/Delete Jobs via UI.
- Rule management interface.
- User settings page.

## Phase 7: Dashboards
- Visualizing `baselines`.
- Success/Fail charts.
- P50/P95 latency trends.

## Phase 8: Differentiators
- Anomaly detection (learned baselines).
- Team collaboration.
- API Tokens for third-party integrations.

---

## Explicit Non-Goals (Until Paid Users Ask)

- No mobile apps
- No AI summaries
- No infra monitoring
- No log aggregation platform
- No workflow orchestration
- No complex "OpsGenie" style on-call rotations
